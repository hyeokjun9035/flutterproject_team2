# 🧩 프로젝트 소개

이 앱은 현재 **날씨를 기반으로 필요한 준비물, 대기질 정보,**
그리고 **내 위치 기준 길 안내·버스 도착 시간을 한 화면에서 제공하는 통합 정보 서비스**입니다.

- 사고나 이슈가 발생시 반경 2km 이내 사용자에게 알림 전송
- 1~5km 선택하여 글쓴 장소 지도로 확인 가능
- 본인이 원하는 장소 저장시 최소 시간 루트 확인 가능

<details>
  <summary>☀️ 메인 페이지 (담당: 김성민, 전경환)</summary>

  <br/>

  <p>
    메인 페이지에서는 “현재 날씨를 한눈에 확인 + 상황에 맞는 행동(이동/제보/커뮤니티)로 빠르게 연결”
    되는 흐름을 목표로 UI/기능을 구성했습니다.
  </p>

  <br/>

  <!-- ===================== -->
  <!-- 1) 날씨 기반 메인 비주얼 -->
  <!-- ===================== -->
  <h3>🌤️ 날씨 기반 메인 비주얼 (낮/밤 + 하늘 + 강수)</h3>
  <p>
    기상청 API(날씨)와 에어코리아 API(대기질)를 활용해 현재 상태를 반영하고,
    하늘 상태(맑음/구름/흐림)와 강수(비/눈), 일출/일몰 기준 낮/밤 여부에 따라
    홈 배경이 자동으로 변하도록 구현했습니다. (전환은 자연스럽게 애니메이션 처리)
  </p>

  <p align="center">
    <img src="https://github.com/hyeokjun9035/flutter_image/blob/main/homepage/home_page1.png" width="210"/>
    <img src="https://github.com/hyeokjun9035/flutter_image/blob/main/homepage/snow.gif" width="210"/>
  </p>

  <ul>
    <li>기상청 API/에어코리아 API 기반 현재 날씨/대기 정보 표시</li>
    <li>날씨/낮밤 기반 동적 배경 테마</li>
    <li>비/눈 등 강수 시 시각적 레이어(효과) 적용</li>
  </ul>

  <br/>

  <!-- ===================== -->
  <!-- 1-1) 준비물/행동 추천 -->
  <!-- ===================== -->
  <h3>🎒 날씨 기반 오늘의 준비물/행동 추천</h3>
  <p>
    오늘의 날씨(기온/강수/바람/체감 등)와 대기 상태를 기준으로
    사용자가 “오늘 챙겨야 할 것”을 빠르게 확인할 수 있도록 준비물/행동 추천을 제공합니다.
  </p>

  <p align="center">
    <img src="IMAGE_URL_CARRY_1" width="210"/>
    <img src="IMAGE_URL_CARRY_2" width="210"/>
    <img src="IMAGE_URL_CARRY_3" width="210"/>
  </p>

  <ul>
    <li>날씨/대기질 기반 준비물(예: 우산/겉옷/마스크 등) 안내</li>
    <li>상황별 행동 추천(예: 외출 주의, 이동 방식 선택 참고 등)</li>
  </ul>

  <br/>

  <!-- ===================== -->
  <!-- 2) 대중교통/즐겨찾기 루트 -->
  <!-- ===================== -->
  <h3>🚌 즐겨찾기 루트 기반 이동 카드 (TMAP 대중교통 + 도보 폴백)</h3>
  <p>
    사용자가 저장한 즐겨찾기(출발/도착)를 기준으로 대중교통 경로를 불러와
    “최소 도보 / 최소 시간 / 최소 환승” 옵션을 요약해서 보여줍니다.
    거리가 너무 가까운 경우에는 대중교통 대신 도보 추천 UI로 폴백 처리했습니다.
  </p>

  <p align="center">
    <img src="https://github.com/hyeokjun9035/flutter_image/blob/main/homepage/route.png" width="220"/>
    <img src="https://github.com/hyeokjun9035/flutter_image/blob/main/homepage/selectroute.png" width="220"/>
    <img src="https://github.com/hyeokjun9035/flutter_image/blob/main/homepage/routeDetail.png" width="220"/>
  </p>

  <ul>
    <li>옵션 선택(최소 도보/시간/환승) + 경로 요약 플로우 표시</li>
    <li>가까운 거리 자동 감지 → 도보 추천 카드로 전환</li>
    <li>상세 경로 보기 화면(RouteView)로 연결</li>
  </ul>

  <br/>

  <!-- ===================== -->
  <!-- 3) 실시간 버스 도착 -->
  <!-- ===================== -->
  <h3>⏱️ 실시간 버스 도착 정보 (지원 노선)</h3>
  <p>
    대중교통 경로에 버스 구간이 포함될 경우, 승차 정류장 기반으로
    실시간 도착 정보를 표시하도록 연동했습니다.
    (지원되지 않는 케이스는 사용자에게 안내 문구로 처리)
  </p>

  <p align="center">
    <img src="https://github.com/hyeokjun9035/flutter_image/blob/main/homepage/route.png" width="230"/>
    <img src="https://github.com/hyeokjun9035/flutter_image/blob/main/homepage/notroute.png" width="230"/>
    <img src="https://github.com/hyeokjun9035/flutter_image/blob/main/homepage/walk.png" width="230"/>
  </p>

  <ul>
    <li>경로 RAW에서 버스 구간 정보 추출 → 정류장 후보 탐색 → 노선 매칭</li>
    <li>주기적 갱신(폴링) + 캐시로 불필요한 호출 최소화</li>
  </ul>

  <br/>

  <!-- ===================== -->
  <!-- 4) 주변 이슈/커뮤니티 연결 + 홈 UX -->
  <!-- ===================== -->
  <h3>📍 내 주변 1km 사건/이슈 + 홈 UX(편집/메뉴)</h3>
  <p>
    현재 위치 기준 1km 반경의 사건/이슈 글을 홈에서 최신 3건까지 미리 확인하고,
    지도 보기/제보/커뮤니티 등 주요 기능으로 빠르게 이동할 수 있도록 구성했습니다.
    또한 홈 카드 순서 편집과 공통 Drawer 메뉴를 통해 사용성이 좋아지도록 했습니다.
  </p>

  <p align="center">
    <img src="https://github.com/hyeokjun9035/flutter_image/blob/main/homepage/routeANDissue.png" width="210"/>
    <img src="https://github.com/hyeokjun9035/flutter_image/blob/main/homepage/nearbyissuemap.png" width="210"/>
    <img src="https://github.com/hyeokjun9035/flutter_image/blob/main/homepage/drawer.png" width="210"/>
  </p>

  <ul>
    <li>내 주변 1km 사건/이슈 Top 3 미리보기 + 지도 화면 연동</li>
    <li>Drawer 메뉴(마이페이지/커뮤니티/제보/설정/로그아웃) 공통 적용</li>
    <li>홈 카드 순서 편집 기능(사용자 설정 유지)</li>
  </ul>
