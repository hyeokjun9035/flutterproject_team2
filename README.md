# 🧩 프로젝트 소개

이 앱은 현재 **날씨를 기반으로 필요한 준비물, 대기질 정보,**
그리고 **내 위치 기준 길 안내·버스 도착 시간을 한 화면에서 제공하는 통합 정보 서비스**입니다.

- 사고나 이슈가 발생시 반경 2km 이내 사용자에게 알림 전송
- 1~5km 선택하여 글쓴 장소 지도로 확인 가능
- 본인이 원하는 장소 저장시 최소 시간 루트 확인 가능

  <h3>🌈 날씨/낮밤 기반 동적 배경</h3>

  <p>
    현재 날씨 상태(맑음/구름/흐림)와 강수(비/눈), 그리고 일출/일몰(낮/밤) 여부에 따라
    메인 페이지의 배경 그라데이션을 자동으로 변경했습니다.
  </p>

  <p align="center">
    <img src="IMAGE_URL_BG_1" width="210"/>
    <img src="IMAGE_URL_BG_2" width="210"/>
    <img src="IMAGE_URL_BG_3" width="210"/>

  <ul>
    <li>낮/밤 + 하늘 상태(SKY) 조합에 따른 그라데이션 테마 적용</li>
    <li>강수(PTY) 발생 시 더 어두운/흐린 톤으로 자동 전환</li>
    <li>화면 전환 시 <code>AnimatedContainer</code>로 자연스럽게 배경이 바뀌도록 처리</li>
  </ul>

  <br/>

  <!-- ===================================================== -->
  <!-- 기능 2: 강수(비/눈) 레이어 -->
  <!-- ===================================================== -->
  <h3>🌧️ 강수(비/눈) 효과 레이어</h3>

  <p>
    비/눈 등 강수 상태일 때 메인 배경 위에 강수 애니메이션 레이어를 추가해
    사용자가 현재 날씨를 직관적으로 느낄 수 있도록 했습니다.
  </p>

  <p align="center">
    <img src="IMAGE_URL_PTY_1" width="210"/>
    <img src="IMAGE_URL_PTY_2" width="210"/>

  <ul>
    <li>강수 상태(PTY) 기반으로 레이어 노출/숨김</li>
    <li>배경과 레이어를 <code>Stack</code>으로 분리해 UI 충돌 없이 적용</li>
  </ul>

  <br/>

  <!-- ===================================================== -->
  <!-- 기능 3: 상단바(TopBar) UX -->
  <!-- ===================================================== -->
  <h3>🧭 TopBar UX 개선 (메뉴/업데이트/새로고침/편집)</h3>

  <p>
    위치 라벨과 업데이트 시각을 한눈에 보이도록 구성하고,
    메뉴(Drawer)·새로고침·카드 순서 편집 기능을 상단바에서 바로 접근 가능하게 했습니다.
  </p>

  <p align="center">
    <img src="IMAGE_URL_TOPBAR_1" width="210"/>
    <img src="IMAGE_URL_TOPBAR_2" width="210"/>
    <img src="IMAGE_URL_TOPBAR_3" width="210"/>
  </p>

  <ul>
    <li>현재 위치 라벨 표시 + 업데이트 시간 표시</li>
    <li>새로고침(<code>RefreshIndicator</code>)와 연동</li>
    <li>카드 순서 편집 UI 진입 버튼 제공</li>
    <li>Drawer 메뉴 버튼 추가 및 공통 AppDrawer 적용</li>
  </ul>

  <br/>

  <!-- ===================================================== -->
  <!-- 기능 4: 카드 순서 편집 -->
  <!-- ===================================================== -->
  <h3>🧩 홈 카드 순서 편집 기능</h3>

  <p>
    사용자가 홈 화면 카드들의 표시 순서를 직접 바꿀 수 있도록 구현했고,
    변경 사항은 유저별로 저장되어 다음 실행에도 유지되도록 했습니다.
  </p>

  <p align="center">
    <img src="IMAGE_URL_ORDER_1" width="210"/>
    <img src="IMAGE_URL_ORDER_2" width="210"/>
    <img src="IMAGE_URL_ORDER_3" width="210"/>
  </p>

  <ul>
    <li>편집 버튼 → 순서 편집 시트 진입</li>
    <li>드래그로 카드 순서 변경</li>
    <li>저장/즉시 반영 (앱 재실행 후에도 유지)</li>
  </ul>

  <br/>

  <!-- ===================================================== -->
  <!-- 기능 5: 즐겨찾기 루트 & 대중교통 요약 -->
  <!-- ===================================================== -->
  <h3>🚌 즐겨찾기 루트 기반 대중교통 경로 요약</h3>

  <p>
    즐겨찾기(출발/도착)를 선택하면 TMAP 대중교통 경로를 불러와
    최소 도보/최소 시간/최소 환승 옵션으로 요약 표시했습니다.
    거리가 너무 가까운 경우는 도보 추천 카드로 폴백 처리했습니다.
  </p>

  <p align="center">
    <img src="IMAGE_URL_TRANSIT_1" width="210"/>
    <img src="IMAGE_URL_TRANSIT_2" width="210"/>
    <img src="IMAGE_URL_TRANSIT_3" width="210"/>
  </p>

  <ul>
    <li>경로 옵션(최소 도보/최소 시간/최소 환승) 선택 UI 제공</li>
    <li>구간별 이동(도보/버스/지하철) 플로우를 한 줄로 시각화</li>
    <li>거리 짧으면 TMAP 호출 스킵하고 도보 추천 UI로 폴백</li>
    <li>경로 보기 버튼으로 상세 RouteView 화면 연결</li>
  </ul>

  <br/>

  <!-- ===================================================== -->
  <!-- 기능 6: 버스 실시간 도착 정보(가능한 노선만) -->
  <!-- ===================================================== -->
  <h3>⏱️ 버스 실시간 도착 정보 연동 (지원 노선)</h3>

  <p>
    대중교통 경로에 버스 구간이 포함된 경우, 정류장 근처 탐색 → 노선 매칭을 통해
    실시간 도착 정보를 표시하도록 구현했습니다.
    (지원되지 않는 케이스는 안내 문구로 처리)
  </p>

  <p align="center">
    <img src="IMAGE_URL_BUS_1" width="210"/>
    <img src="IMAGE_URL_BUS_2" width="210"/>
    <img src="IMAGE_URL_BUS_3" width="210"/>
  </p>

  <ul>
    <li>경로 RAW에서 버스 구간의 승차 정류장 좌표/노선번호 추출</li>
    <li>주변 정류장 후보를 조회하고, 실제 노선 도착정보가 뜨는 정류장을 선택</li>
    <li>1분 주기 폴링 + 캐시로 불필요한 호출 감소</li>
    <li>지원 불가/정류장 미탐색 시 사용자 안내 문구 표시</li>
  </ul>

  <br/>

  <!-- ===================================================== -->
  <!-- 기능 7: 내 주변 1km 사건/이슈 미리보기 -->
  <!-- ===================================================== -->
  <h3>📍 내 주변 1km 사건/이슈 (Top 3)</h3>

  <p>
    현재 위치 기준 1km 반경의 “사건/이슈” 게시글을 최신순으로 필터링해
    홈에서 3건까지 빠르게 확인할 수 있도록 구현했습니다.
  </p>

  <p align="center">
    <img src="IMAGE_URL_NEARBY_1" width="210"/>
    <img src="IMAGE_URL_NEARBY_2" width="210"/>
    <img src="IMAGE_URL_NEARBY_3" width="210"/>
  </p>

  <ul>
    <li>Firestore 최신 글 후보를 가져온 뒤, 거리(haversine)로 1km 필터링</li>
    <li>좋아요/댓글 수 등 메타정보 함께 표시</li>
    <li>글 탭 시 상세 페이지 이동, 지도 보기 버튼으로 Nearby Map 화면 이동</li>
  </ul>
