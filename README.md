# 🧩 프로젝트 소개

이 앱은 현재 **날씨를 기반으로 필요한 준비물, 대기질 정보,**
그리고 **내 위치 기준 길 안내·버스 도착 시간을 한 화면에서 제공하는 통합 정보 서비스**입니다.

- 사고나 이슈가 발생시 반경 2km 이내 사용자에게 알림 전송
- 1~5km 선택하여 글쓴 장소 지도로 확인 가능
- 본인이 원하는 장소 저장시 최소 시간 루트 확인 가능
