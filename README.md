# 🧩 프로젝트 소개

이 앱은 현재 **날씨를 기반으로 필요한 준비물, 대기질 정보,**
그리고 **내 위치 기준 길 안내·버스 도착 시간을 한 화면에서 제공하는 통합 정보 서비스**입니다.

- 사고나 이슈가 발생시 반경 2km 이내 사용자에게 알림 전송
- 1~5km 선택하여 글쓴 장소 지도로 확인 가능
- 본인이 원하는 장소 저장시 최소 시간 루트 확인 가능

  <h3>📌 한눈에 보기</h3>

  <ul>
    <li><b>현재 위치 기반 대시보드</b> (위치 권한/서비스 상태에 따라 안전한 폴백 처리)</li>
    <li><b>날씨·시간대(낮/밤) 기반 배경</b> + 강수(비/눈) 상태에 따른 화면 분위기 변화</li>
    <li><b>카드형 홈 구성</b> + <b>유저별 카드 순서 편집</b> 및 저장</li>
    <li><b>새로고침(Refresh)</b> + 로딩/에러 상태 UI 처리</li>
  </ul>

  <br/>

  <h3>🌤️ 날씨/환경 정보</h3>
  <ul>
    <li><b>현재 날씨 요약</b>: 온도/하늘상태/강수 여부 등 핵심 정보 제공</li>
    <li><b>시간대별(단기) 예보</b>와 <b>주간 예보</b> 제공</li>
    <li><b>특보/경보 배너</b>: 기상 경보가 있을 경우 상단에 강조 표시</li>
    <li><b>대기질(미세먼지 등) 정보</b> 표시 + 지역 라벨(역지오코딩) 연동</li>
  </ul>

  <br/>

  <h3>🧠 날씨 기반 추천 기능</h3>
  <ul>
    <li><b>행동 추천</b>: 날씨(강수/기온/하늘 상태)에 따라 오늘 추천 행동 제시</li>
    <li><b>준비물/주의사항 추천</b>: 우산/겉옷/마스크 등 날씨 기반 준비물 안내</li>
    <li><b>강수 시작 알림/주의</b> 등 상황 기반 안내 문구 제공(조건 충족 시)</li>
  </ul>

  <br/>

  <h3>🧭 이동/경로 카드 (즐겨찾기 기반)</h3>
  <ul>
    <li><b>즐겨찾기 루트</b> 기반으로 출발→도착 경로 요약 제공</li>
    <li>경로 옵션 제공:
      <ul>
        <li><b>최소 도보</b> / <b>최소 시간</b> / <b>최소 환승</b> 선택</li>
        <li>각 옵션의 <b>총 소요시간·도보시간·환승 횟수</b> 요약</li>
      </ul>
    </li>
    <li><b>경로 상세 보기</b>: Route View로 이동해 구간별 이동(도보/버스/지하철) 확인</li>
    <li><b>거리 매우 짧은 경우 도보 추천 폴백</b>: TMAP 호출을 스킵하고 도보 이동 카드로 안내</li>
  </ul>

  <br/>

  <h3>🚌 실시간 버스 도착 정보</h3>
  <ul>
    <li><b>TMAP 경로의 버스 구간</b>에서 노선/정류장 정보를 추출해 도착 정보 조회</li>
    <li><b>1분 단위 폴링</b>으로 도착정보 갱신 + 마지막 업데이트 시각 표시</li>
    <li><b>정류장 후보 검색 → 실제 노선 도착정보가 뜨는 정류장 자동 매칭</b></li>
    <li><b>정류장 캐시(10분)</b>로 불필요한 API 호출 감소</li>
    <li><b>지원하지 않는 노선/정류장 케이스</b>는 안내 문구 표시(반복 실패 시 폴링 중지/쿨다운 처리 등)</li>
  </ul>

  <br/>

  <h3>📍 내 주변 1km 사건/이슈</h3>
  <ul>
    <li><b>내 위치 반경 1km</b> 이내의 “사건/이슈” 게시글 <b>최신 3건</b> 표시</li>
    <li>각 글의 <b>거리(m)</b>, <b>좋아요 수</b>, <b>댓글 수</b>, <b>작성 시간</b> 요약 제공</li>
    <li><b>지도 보기</b>: 주변 이슈를 지도 페이지로 이동해 확인</li>
    <li><b>제보</b>: 제보 페이지로 바로 이동</li>
    <li><b>실시간 갱신</b>: 댓글/좋아요 변화가 홈 카드에도 반영되도록 스트림 기반 업데이트</li>
  </ul>

  <br/>

  <h3>🧩 홈 화면 UX/시스템</h3>
  <ul>
    <li><b>Drawer 메뉴</b> 제공: 홈/커뮤니티/지도/마이페이지/설정/로그아웃 등 빠른 이동</li>
    <li><b>로그인 상태/권한 상태</b>에 따라 안전하게 UI 노출(예: user null, 좌표 null 처리)</li>
    <li><b>캐시 활용</b>:
      <ul>
        <li>대시보드 데이터 캐시로 탭 왕복 시 빠른 표시</li>
        <li>역지오코딩 결과 캐시로 반복 호출 방지</li>
      </ul>
    </li>
    <li><b>에러/로딩 UX</b>: 초기 로딩, 실패, 재시도 버튼 등 상태별 화면 구성</li>
  </ul>
